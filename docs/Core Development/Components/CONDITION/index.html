CaseDescription
IFUse if to execute a sub flow, if a specified condition is true
ELSEUse else to execute a another sub flow, if the same condition is false
ELSEIFUse else if to execute a another sub flow, if the first condition is false
LOOPUse Loop to iterate over nodes /elements inside sub flow
ANDUse And to execute a subflow if all statements are true
ORUse Or to execute a subflow if one of the statements is true
